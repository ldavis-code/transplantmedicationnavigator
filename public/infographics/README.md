# Infographics

This folder contains educational infographics for the Transplant Medication Navigator.

## Contents

- `financial-assistance-comparison.png` - Visual comparison of PAP, Foundation, and Copay Card assistance options

## Image Guidelines

- **Format**: PNG or SVG preferred
- **Mobile width**: 400-600px
- **Desktop width**: 800-1200px
- **Alt text**: Always provide descriptive alt text for accessibility

## Usage

Reference images in React components:

```jsx
<img
  src="/infographics/financial-assistance-comparison.png"
  alt="Comparison of PAP, Foundation, and Copay Card assistance options"
/>
```
